# Data Structures intro

## What do the do?

- Data structures are collections of values, the relatioships among them and the functions or operations that can be applied to the data.

## Why so many data structures?

- Some are highly specialized, while others are generally more used. Different data structures excel at different things.

## Why care?

- The more time you spend as a developer, the more likely you'll need to use one of those data structures.

-You've already worked with many of them unknowingly.
